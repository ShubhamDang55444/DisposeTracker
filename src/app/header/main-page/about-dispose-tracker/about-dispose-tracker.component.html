Dispose Tracker

A Portal to analyze PortalProfiler Logs

As we Know, PortalProfiler allows tracking of .net Applications and used for used for detecting of non-disposed disposable instances. It allows user to save summary of non-disposed objects since start of application in a NotePad file. Sometimes it becomes difficult sometimes to analyze the NotePad File and that's where Dispose Tracker makes it easy.

Dipose Tracker allows user to upload the summary file of Portal Profiler and analyze it in efficent way.
